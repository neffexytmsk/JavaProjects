# Система управления гостиничными номерами (Hotel Room Management)

Десктопное приложение на Java для управления и классификации номеров в гостинице. Программа демонстрирует объектно-ориентированный подход с наследованием, инкапсуляцией и возможностью сортировки данных.

## 📋 Описание предметной области

Приложение моделирует работу гостиничного фонда. В основе иерархии лежит базовый класс `Room` (Комната), от которого наследуются три типа номеров с различными характеристиками и функционалом.

### Класс `Room` (Родительский класс)
Базовый класс, описывающий общие свойства любого номера:
- **Id** — уникальный идентификатор номера
- **codeNumbers** — код номера (например, "101", "A42")
- **numberPeople** — максимальное количество человек для размещения
- **price** — стоимость за сутки

### Дочерние классы

1. **`StandardRoom` (Стандартная комната)**  
   Наследует все свойства родителя без добавления новых. Представляет базовый тип номера.

2. **`SemiLuxRoom` (Комната полулюкс)**  
   Наследует свойства родителя. Отличается повышенным уровнем комфортности (`comfortType`), который определяется как "Полулюкс".

3. **`LuxRoom` (Комната люкс)**  
   Помимо свойродительских свойств, добавляет новые:
   - **minRentPeriod** — минимальный срок сдачи (в днях)
   - **maxRentPeriod** — максимальный срок сдачи (в днях)

## ✨ Функциональные возможности

- **Создание объектов** различных типов номеров с помощью конструкторов
- **Инкапсуляция данных**: все поля классов закрыты (`private`), доступ осуществляется через геттеры и сеттеры
- **Информативный вывод**: переопределённый метод `toString()` форматирует вывод полной информации об объекте
- **Хранение данных**: специализированный класс `RoomManager` для управления списком номеров
- **Добавление номеров**: метод для включения нового номера в общий список
- **Печать списка**: метод для вывода всех номеров в читаемом формате
- **Сортировка**:
  - По цене (`price`)
  - По количеству мест (`numberPeople`)

1. Убедитесь, что на вашем компьютере установлена Java Runtime Environment (JRE) версии 8 или выше.
